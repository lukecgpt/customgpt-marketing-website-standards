The Two Hard Rules

These override everything. If a design breaks either rule, it is off-brand.

Shadow
Always no shadow. Nothing on the site casts a shadow.
Radius
Always no radius. Square corners everywhere.

03

Buttons & CTAs

Every CTA is either primary (filled indigo) or secondary (ghost / outlined). In any two-CTA group: exactly one primary, one secondary. The secondary is always ghost — chosen by background, not by action type. No button ever carries an arrow (arrows belong only to inline text links).

Consistent sizing: a primary button is the same size whether it stands alone or sits in a button group. A lone primary is never larger than a paired one. Pricing pattern: the featured ("Most Popular") plan uses the filled primary; every other plan CTA is a ghost secondary — even when the action is "Try free for 7 days." Shape: all buttons are square (no radius), no shadow, 1.5px border, hover lifts translateY(-1px).

07

Design Audit Checklist

Run this against any page. A page passes only if every line is true.

  • No shadows. Nothing on the page casts a shadow.
  • Square corners. No rounded corners anywhere (cards, buttons, inputs, avatars, images).
  • Fonts: Lora for every heading and quote; Inter for everything else. No third typeface.
  • Headings use .h1 / .h2 / .h3 classes — no custom font-size-only heading classes.
  • Colors all map to a token. No stray hex values outside the palette above.
  • Background is warm off-white #FAF9F6 — not pure white #FFFFFF.
  • Borders are warm tan #B8B2A8 at 1.5px — not gray #E5E7EB. Bottom-border only on sections.
  • CTAs: one primary + one secondary per group; secondary is always ghost; chosen by background.
  • Primary size is consistent — a lone primary is no larger than a paired one.
  • No arrows on buttons. Arrows only on inline .text-link discovery links.
  • Icons are Lucide inline SVG, 22px in a 44px box, tints rotating indigo → pink → lavender.
  • Spacing values sit on the 4px scale (4 / 8 / 12 / 16 / 24 / 32 / 48).
  • No em dashes in any copy. Friction line reads "7-day free trial · Cancel anytime", never "No credit card".
